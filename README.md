## 🧾 Overview

A multi-role parcel management platform with real-time tracking, smart logistics workflow, and analytics-driven control system.

This system is designed to simulate a real-world courier service with Admin, Rider, and User panels, enabling complete end-to-end parcel delivery management.


---


## 🎯 Key Highlights

* 📦 **Real-time parcel tracking system**

* 🧭 **Smart delivery assignment based on location**

* 📊 **Analytics-driven admin dashboard**

* 👥 **Multi-role access system (Admin / Rider / User)**

* 🧾 **Automated invoice generation**

* 🗺 **Branch & coverage map system**

* 💳 **Secure payment with stripe**


---


## 👨‍💼 Admin Panel

Central control dashboard for full system management.

### Features:

* 📊 **Live overview of parcels, users, revenue, and delivery status**

* 🚴 **Rider onboarding approval system with verification**

* 👤 **User management (role update, block/unblock, pagination)**

* 🧠 **Smart delivery assignment based on location**

* 📦 **Parcel monitoring, cancellation, invoice generation & export**

* 📈 **Rider performance analytics**

* 🎫 **Support ticket management**


---


## 🚴 Rider Panel

Dashboard for managing delivery operations.

### Features:

* 💰 **Earnings tracking dashboard**

* 📬 **Accept or reject delivery tasks**

* 🔄 **Delivery workflow (Pickup → In Transit → Delivered)**

* 📜 **Delivery history & performance stats**

* 💬 **Communication with admin support**
